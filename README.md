## Run Locally

Clone the project

```bash
  git clone https://github.com/Deeptarko/Blog-App.git
```

Go to the project directory

```bash
  cd blog-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
